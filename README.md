# 🏆 Fanatics AI Support Bot: Proof of Concept
**Live Demo:** [Check out the live bot here!](https://fanatics-support-bot-d9hvnawzyyvzsfa7wxappqt.streamlit.app/)

## 🎯 The Mission
Fanatics processes thousands of repetitive customer inquiries daily. This project is a **Retrieval-Augmented Generation (RAG)** chatbot designed to deflect low-complexity tickets regarding returns, shipping, and sizing, allowing human agents to focus on high-value interactions.

## 🛠️ Tech Stack
* **AI Engine:** Google Gemini 3 Flash (via Google AI Studio)
* **Framework:** Python & Streamlit
* **Architecture:** RAG (Retrieval-Augmented Generation)
* **Data:** Official Fanatics Public Support Policies

## 💡 Key Features
* **Zero Hallucination:** The bot is grounded strictly in the provided `knowledge_base` text files.
* **Domain-Specific Logic:** Handles complex rules like "365-day returns" and "International VAT/Duties."
* **Cost Efficient:** Built using high-performance, low-cost AI infrastructure.
